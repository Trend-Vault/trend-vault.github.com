# trend-vault.github.com# Trend-Vault - Poster Store Website

A modern, dark-themed e-commerce website for selling posters (Anime, Car, Bike, and more) with size options (A3/A4) and special bundle offers.

## Features

- 🎨 **Dark Theme** with cyan/purple/pink gradient accents
- 📱 **Fully Responsive** design for mobile, tablet, and desktop
- 🛒 **Shopping Cart** with localStorage persistence
- 💰 **Bundle Offer**: Buy 3 A3 posters for ₹99 (was ₹120)
- 📏 **Size Options**: A4 (₹20) and A3 (₹40)
- 🔍 **Category Filters**: Anime, Car, Bike, Other
- 💾 **Cart Persistence**: Items saved even after page refresh
- 🎯 **Toast Notifications** for user actions
